# 프로젝트 세팅 방법

### 1. Https Key 받기

Https key를 받아 서버의 `/etc/letsencrypt/live/도메인 이름`에 저장합니다
만약 도메인 이름이 http://i5b101.p.ssafy.io/와 다르다면 `/nginx/default.conf`파일의 저장 경로를 수정해야합니다.

### 2. nginx, frontend, backend 실행

`docker-compose up -d`
